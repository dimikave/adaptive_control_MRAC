# Καβελίδης Φραντζής Δημήτριος - 9351 - Εργασία 2η
## Model Reference Adaptive Control

Σε αυτόν τον φάκελο υπάρχουν τα παραδοτέα της δεύτερης εργασίας
για το μάθημα Ευφυή και Προσαρμοστικά Συστήματα Αυτομάτου Ελέγχου (ΕΠΣΑΕ).

Συγκεκριμένα:
- Main_1.m			- Περιέχει το βασικό πρόγραμμα που βγάζει τα
				  αποτελέσματα της εργασίας.
- IdealSys.m			- Περιέχει την odefunc μας για την διαφορική
				  εξίσωση του ιδανικού συστήματος
- LinUnSys.m 			- Περιέχει την odefunc για την διαφορική που
				  περιγράφει το σύστημά μας παρουσία
				  (γνωστών) αβεβαιοτήτων, με σκοπό την παρουσίαση
				  αστάθειας.
- mrac_sys.m			- Περιέχει την odefunc για την διαφορική που
				  πριγράφει το σύστημά μας παρουσία άγνωστων
				  αβεβαιοτήτων εφαρμόζοντας ΠΕΜΑ (MRAC).
- RefSig.m			- Συνάρτηση που περιγράφει το σήμα αναφοράς μας.

- Assignment_2_9351.pdf		- Γραπτή αναφορά / παρουσίαση των αποτελεσμάτων
